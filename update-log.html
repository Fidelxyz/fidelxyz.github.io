# V4.0 (Hexo + Github Pages + Shoka)

# 2021-05-06

  • Hexo 版本从 5.3 升级到 5.4。

# 2021-04-28

  • 使用了 VSCode 的 markdownlint 插件，规范文章 Markdown 格式。（VSCode 针不戳！）

# 2021-03-14

  • Polyfill 源更换为 Alicdn 的镜像。
  • Google Font 源更换为 fonts.dogedoge.com 的镜像。

# 2021-02-15

  • 为图片添加了 max-height 属性，优化长图 / 手机截图阅读体验。
  • 去除了 CDN 资源路径末尾的版本号，修复了可能导致 CDN 未及时刷新的问题。

# 2020-01-30

  • 修复了 Archive 和 Tags 部分页面由于 block:mixed-content （https 页面发送 http 请求被阻止）而无法打开的问题。

# 2020-01-28

  • 修复了 Sidebar 图标显示错误。
  • 移除了插件 hexo-toc，原因是与 Shoka 主题使用的 markdown-it-toc-and-anchor 会产生冲突。
  • 修复了 Header nav 显示透明的问题。

# 2020-01-24

  • 更换主题：hexo-theme-shoka。

    Shoka 主题真的太漂亮了！

新增亮点：

  • lozad.js，支持懒加载（lazyload），大幅提升访问速度。
  • hexo-feed，支持 3 种 Feed 格式。
  • pangu.js，中文和英文、數字、符号之间自动插入空格。
  • hexo-renderer-multi-markdown-it, 压缩 css/js/html，漂亮的代码块渲染以及各种拓展功能。
  • hexo-algoliasearch，支持站内搜索。
  • pjax，无刷新加载页面。
  • pace，加载进度条。
  • quicklink，链接资源预加载。
  • KaTeX，高效的数学公式渲染。
  • 添加了 Donate 二维码。

修改内容：

  • 增加了对 Gitalk 的支持，支持懒加载（lazyload）。
  • 关闭了随机封面图。
  • 图片部分更换为 WebP 格式，优化加载速度。
  • Polyfill.js 从 v2 升级至 v3。
  • 使用了 loli.net 的 Google Fonts 加速。

  • 增加代码块 margin-bottom ，优化阅读体验。

# 2020-08-30

  • 将评论系统从 Valine 更换为 Gitalk。

    放弃 Valine 的原因有两个，第一，Valine 不需要验证个人信息，我无法确定评论者身份的真实性；第二，Valine 的自动邮件回复可能会被滥用，通过我的邮箱发送垃圾邮件。

    即便如此，Valine 依旧是一个出色的评论系统。

    在 Fidel's Lab 使用 Valine 的 2 年零 5 个月里，Valine 及 Leancloud、 Valine-Admin 共承载了 13 位评论者的 36 条评论。

# 2020-04-14

    其实原本想使用 font-face 强制加载 PingFang （Windows 的默认中文字体 Microsoft YaHei 实在是太丑了），但由于中文字体应用 font-face 需要的优化极其繁琐，因而弃用。

  • 在文章的 footer 右下角添加了「最近更新于」一栏。

  • 将 Blog 源代码上传到 GitHub (Private Repository)，保证数据安全。

  • 移除了右上角没啥用的 Search。
